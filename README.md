# Onboarding Summit Map 2.5

北美新教师 Onboarding 培训地图（React 19 + TypeScript + Vite）。

## 文档

- **[PROJECT_MAP.md](./PROJECT_MAP.md)** — 代码与文件对照（推荐 Agent / 新同学首读）
- **[AGENTS.md](./AGENTS.md)** — Cursor Agent 工作说明

## 快速开始

```bash
npm install
npm run dev -- --port 5179 --strictPort
```

| URL | 页面 |
|-----|------|
| `/portal-home` | 教师门户 |
| `/launch-map` | Summit 培训地图 |

## 从旧仓库迁移

本目录由 `na-new-teacher-map` 复制整理，版本号 **2.5.0**。不含 `node_modules` / `dist`，首次需 `npm install`。
